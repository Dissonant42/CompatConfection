This mod adds/will add compatibility and parity between The Confection REBAKED and some other mods, namely Thorium, Mech Bosses Rework, Spirit Reforged and Infected Qualities. 
This includes (but is not limited to):
- Neapolinite gear and helmets for other classes
- Jungle/Savannah Confection sub-biomes
- Alternative materials
- Recipe changes
- More block conversions for solutions (gems, mosses, mud, coarse dirt)
- Confection Plantera+Life Fruit sprites?
- Loot changes
